Auto printer

Impression automatique depuis un navigateur web destiné à une imprimante thermique de 72mm.

Solution trouvée ici : https://stackoverflow.com/a/70289151
Pour déclencher une impression à partir de JavaScript, il existe la fonction native `window.print()`. Cette fonction ouvre une fenêtre de dialogue d'impression du navigateur, permettant de configurer l'imprimante (par exemple, retirer les marges par défaut). Une fois correctement configurée, nous pouvons modifier une option par défaut du navigateur (en l'occurrence Firefox) pour supprimer la fenêtre de dialogue d'impression et imprimer directement le document en utilisant l'option `silent_print`.

Ensuite, la requête de média CSS `@media print` nous permet d'appliquer un style uniquement lors de l'impression. Cela nous permet de masquer le contenu de la page, sauf pour l'élément à imprimer.
